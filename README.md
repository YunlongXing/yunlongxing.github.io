# Yunlong Xing Homepage

Personal homepage for Yunlong Xing, built with Jekyll and hosted on GitHub Pages.

## Local Development

The lockfile currently expects Bundler 2.2.19:

```sh
gem install bundler:2.2.19
bundle _2.2.19_ install
bundle exec jekyll serve
```

Then open `http://127.0.0.1:4000/`.
